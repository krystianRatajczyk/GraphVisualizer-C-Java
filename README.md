## Wizualicja grafu planarnego
Interaktywna aplikacja desktopowa napisana w języku Java (Swing), służąca do wizualizacji, edycji i analizy grafów planarnych. Projekt pozwala na dynamiczne zarządzanie strukturą grafu oraz uruchamianie algorytmów grafowych z poziomu intuicyjnego interfejsu użytkownika.
